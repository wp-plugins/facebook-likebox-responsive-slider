== Description ==

[Demo](http://innovativeplugins.com) | [Facebook Comment Slider](http://sympies.com/facebook-comment-slider/)

 Facebook LikeBox Slider helps you to get more likes for your Facebook Page with displaying a responsive like box on your home page of your website. Simple to use and it has a lot of customization options. Includes themes, style (dark & light), option to force users to like your site and unlock the content, you can find much more available settings.
 
  = Lock Screen =
 With Lock option, you can force your visitors to hit on the like button, because the plugin able to lock the screen until the visitor likes your page.
 
  = Responsive =
 The visitors can enjoy your Like Box on any screen, including Tablets and Mobiles.
 
  = Flexible Positions =
 It is possible to set the vertical position in distance and put the Like Box on the left or on the right side of the screen.
 
  = Languages =
 The Facebook Like Box Responsive Slider supports more than 70 languages, sure you will find the best that can fit to your website.

== Installation ==

- First, log in to the Administration area of your Wordpress website and select Plugins.
- Click Add New on the top of the page
- Select Upload on the top of the page again
- Browse the plugin zip file from your computer and click Install Now.
- Click on the Activate Plugin link and the plugin will be active.
- After successful plugin activation, you will redirect to the plugins' list page.
- You can find the Facebook LikeBox Slider plugin in the list.
- Set up the plugin in the Settings/FB LikeBox Slider.

== Frequently Asked Questions ==

= Do I need a Facebook App ID to use the plugin? =

Officially yes, but the plugin able to work without that, however using the Like Box wihtout the App ID can causing conflicts on your webite. You can find a step-by-step guide in the Documentation about how to create Facebook App ID.

= How can I add Like Box to all of my pages? =

This free plugin allows you to display the Like Box Slider on the Home Page only, you can purchase the [Pro Version - Facebook Comment Slider](http://sympies.com/facebook-comment-slider "Facebook Comment Slider") to unlock this option and use many of other features. (including Comment Box and Fan Page Wall)
