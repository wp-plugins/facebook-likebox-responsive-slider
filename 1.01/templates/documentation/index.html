INTRODUCTION

The plugin provides a Facebook Like Box on the Home page of your website with several options. You can check the full list of settings below. Facebook LikeBox Slider is the free version of Facebook Comment Slider, that's contains a Comment Box with Share buttons, Like Box and a Fan Page Wall with unlimited usage on all pages/posts.

FEATURES

- fully responsive

- ability to lock the screen until the user like your website

- fixed slider position

- automatic slide-in when the visitor reach the bottom of the page

- you can enable or disable the user can close the slider

- ability to set up timer to close the slider

- you can specify your own notice

- fully localizable like box, choose from more than 70 languages

- selectable direction of the slider

- you can set up to open the slider automatically just once per user

- fancy jquery animation with fadeout and slide effect

- detailed documentation

- transparent background when the screen locked

- unlock screen with click on the background (if you allow to close the slider for the users)

- full customizable separated CSS file

- you can hide or show the facebook icon

- adjustable transparency

- three different icon size

- auto open the slider if you want, when the users reach the bottom of the page

- mobile-friendly slider

CREATE FACEBOOK APP ID

Before you use the plugin, you have to set up the Application ID.

Make sure, you logged in with Personal and verified account on facebook.com
First, visit the following page and create a new application:

https://developers.facebook.com/apps

Click on the Add a New App link and select the www option

Type any name for your App, it doesn't matter, then choose the Create New Facebook App ID

Select a category and you can now create your Facebook App:

On the next screen you can see your App ID and you can specify your website URL:

Go to https://developers.facebook.com/apps again, select your app and click on the Settings at the left side menu to specify your email address. It is required to activate your App.

After you save your email address with Save Changes button, choose the Status & Review menu, disable the SandBox mode (activate your App).

You can see the App ID on the top part of the page in the Settings menu. You have to save this ID to the settings of the plugin.
The facebook works with cache, so you may need wait some time to going live. It can take from few minutes to even few hours, depends on the Facebook.

SETTINGS

The pugin has a simple, but powerful options.

You can reach the settings by clicking on the Settings on the left side and choose the Facebook LikeBox Slider. Or you can also visit the settings page if you choose the Plugins, Installed Plugins and select the Settings under the Facebook LikeBox Slider in the Plugin list page.


Facebook App ID is required to get a normal visibility, however the Like Box can work without that, the Facebook initialization trying to use the Facebook App ID, so it isn't official to use the Like Box without that. Create a Page on Facebook and fill the Facebook Page Name field. Pay attention to use only the name, not the full URL of your Facebook Page.

You can set to invisible the Facebook icon if you choose the ON option Hide icon setting. By default, this value is on, but you can combine with the auto open setting on the different tabs.

The LikeBox Slider can appear automatically when the user almost reach the bottom of the page. If you want this, please check the ON option under the Auto open at bottom of the page

(See images below) With lock screen option, you can enable a transparent background, that is lock the screen and the users can't click on anything, except the comment slider.

If you would like to allow to close the slider, you can do it with User can close the slider option. The slider closeable by several ways:

  • user click on the x button at the bottom of the slider
  • user click on the facebook icon
  • user click on the transparent background


You can specify an exact transparency value of the faded-in background in percentage. If you set 100, the background will be fully black.

The slider will disappear autimatically, when you set a valid number to the timer and the time is up. If you set zero, the slider will disapper only if the user leave a comment or close the slider.

You can specify the message on the Like Box, this will be displayed on the slider's top area.

By default, the slider come on the left side of the screen, but you can choose the another side, if you want with the direction option.

The plugin can track the users, who already saw the slider. If you enable display once per user, the slider will appear automatically only once for every users. (except they delete cookies)

The Like Box fully localizable by the Language parameter. You can choose from exactly 77 languages, these are the full list of Facebook's languages.


Style of the Facebook LikeBox Slider is available in dark or light color variations. You can change it easily with the Style attribute.

UPDATE: With Disable on mobile option you can disable globally the entire plugin on devices below 800x600 resolution.

You can disable / set time to the shake effect (choose from 5 sec to 5 min.) or select the HeartBeat effect.

Hide Likebox for already Liked option disable the LikeBox for users who already liked your website. This settings using Cookies, if the visitor delete the cookies from the browser, the LikeBox will be visible again. (cookies are expired after 999 days)

Skin options allows to select templates for all sliders from the inbuilt skins of the plugin.

It is possible to open or close any of the slider with a simple link. You need to only add a specific class to do that. Let see the full list:

  • classname: .fcs_open_likebox, method: open the LikeBox Slider, example: <a href="#" class="fcs_open_likebox">link text</a>
  • classname: .fcs_close_likebox, method: close the LikeBox Slider, example: <a href="#" class="fcs_close_likebox">link text</a>
  • classname: .fcs_close, method: close the last opened slider, example: <a href="#" class="fcs_close">link text</a>
  • classname: .fcs_hide, method: hide all sliders, example: <a href="#" class="fcs_hide">link text</a>
  • classname: .fcs_show, method: show all hidden sliders, example: <a href="#" class="fcs_show">link text</a>
